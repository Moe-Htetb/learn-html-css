# Module 01: HTML Fundamentals & Semantic Structure

HTML (HyperText Markup Language) is the backbone of every website. It defines the structure and meaning of your web content.

## 1. Document Structure

Every HTML5 document starts with a standard boilerplate:

- `<!DOCTYPE html>`: Tells the browser to use the latest HTML standard.
- `<html>`: The root container for all other elements.
- `<head>`: Contains metadata, character encoding, and links to external styles/scripts.
- `<body>`: Contains the visible content of your website.

## 2. The Power of Semantic HTML

Instead of using `<div>` for everything, we use Semantic Tags to give meaning to our code. This improves **SEO** and **Accessibility**.

- `<header>`: Introductory content or navigation links.
- `<nav>`: A section intended for navigation.
- `<main>`: The unique, primary content of the document.
- `<section>`: A thematic grouping of content.
- `<article>`: Self-contained composition (like a blog post).
- `<footer>`: Information about the author, copyright, or contact info.

## 3. Essential Elements

- **Headings (`h1`-`h6`)**: Hierarchical titles for content.
- **Paragraphs (`p`)**: For blocks of text.
- **Anchors (`a`)**: To create hyperlinks using the `href` attribute.
- **Lists (`ul`, `ol`, `li`)**: For organizing items or navigation menus.

## Included Features Checklist

- Semantic HTML (`header`, `main`, `section`, `article`, `aside`, `footer`)
- CSS selectors (element, class, id, pseudo-class, pseudo-element targets)
- Specificity examples (`#site-header`, `.menu a`, `.btn`)
- Full box model usage (padding, border, margin, border-radius, box-sizing)
- Units (`px`, `%`, `rem`, `clamp`, `vh`-style spacing behavior)
- Typography (font stack, line-height, heading scale)
- Layout with Flexbox (`.nav`, `.hero-actions`, `.footer-layout`)
- Layout with Grid (`.hero-layout`, `.feature-grid`, `.pricing-grid`)
- Table styling (`.course-table`)
- Form styling (`input`, `select`, `textarea`, focus-visible)
- Hover/focus states for better UI and accessibility
- Responsive media queries for tablet and mobile
